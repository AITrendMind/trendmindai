Privacy Policy

Last updated: November 2025

1. Overview

TrendMindAI respects user privacy and complies with TikTok’s Developer Policy. This Privacy Policy explains what data is handled and how.

2. Data Collection

TrendMindAI does not collect or store personal information from users. The application uses TikTok’s official OAuth 2.0 authentication only to obtain temporary access tokens required for posting content through the TikTok API.

3. Data Usage

Any TikTok data accessed via API (such as basic profile information or media upload status) is used exclusively for demonstration and development within TikTok’s sandbox environment.

4. Security

All communication with TikTok servers is encrypted via HTTPS. Tokens are handled locally and are never stored on external servers.

5. Contact

If you have any privacy concerns or questions, please contact: aitrendmind@gmail.com
